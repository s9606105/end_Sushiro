# 109111134 林禹廷
## 動機
```
由於我們想做一個有關資料庫的主題，在我們一番熱烈討論後，
投票選擇一個有insert into, select等sql語法的壽司郎；
這個網頁有許多的css 去做一個觸碰的效果，像是其中有滑過後會顯示了解詳情，
以及有輪播的效果，加上還有店鋪募集可以用來postback資訊到另一個網站，
或是menu去控制看的內容等，可以學習到的內容可以說是非常多元，
有css、javascript、csharp、html等，
所以壽司郎成為了我們這學期的期末報告網站。
```
## 解決方法與設計規劃
```
1. 在做主板時，遭遇到最大的困難，無非就是壽司郎的logo、top往上滑動的問題、
nav bar的縮放問題，為了實現RWD，去運用到許多@media控制其效果，
以及改變其原本的屬性值，來做到視覺的提升。
2. 第二個在做Shop時，遇到了另一個問題，就是如何運用csharp去投放想要的資料，
以及運用csharp將資料傳到另一個介面，以達到搜尋後的資料保留。
3. 再來是在切版是遇到的問題，需要考慮到每個所需的版面數量，
並切出要更改與保留的內容。
```
